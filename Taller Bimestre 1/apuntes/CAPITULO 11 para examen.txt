 se muestran a continuación. Observe que solo mediante 1 AND 1 se obtiene 1. Cualquier otra combinación da como resultado un 0.
 1 Y 1 = 1
0 Y 1 = 0
1 Y 0 = 0
0 Y 0 = 0

Nota: En la lógica digital, 1 representa True y 0 representa False. Cuando se utiliza una operación AND, ambos valores de entrada
 deben ser True (1) para que el resultado sea True (1).
 eso explica el porque 1 y 0 da como resultado 0
